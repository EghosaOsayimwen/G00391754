LEDLenser P7 Core Flashlight
LEDLenser P7 Core is a compact flashlight running on common AAA batteries. With adjustable focusing, the P7 Core is a great choice for up-close and distance lighting applications. The front bezel easily slides forward or back to adjust from spot to flood beam.
  • Output: 450, 250 or 40 lumens
  • Battery Type: AAA alkaline (4), included
  • Run Time: 1 to 25 hours
  • Beam Range: 984 feet (300 meters)
  • Dimensions: 5.18″ long x 1.45″ bezel diameter x 1.16″ body diameter
  • What’s Included: LEDLenser P7 Core flashlight, batteries, wrist lanyard
